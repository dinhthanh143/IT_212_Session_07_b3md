## Phần 2: Chi tiết nội dung 3 lượt Prompt

### Vòng 1 (Robustness)

> Bạn hãy đóng vai trò là chuyên gia Java Senior. Dưới đây là mã nguồn cũ cần refactor:
>
> ```java
> public class ReportGenerator {
>     public void gen(String f, String out) throws Exception {
>         BufferedReader r = new BufferedReader(new FileReader(f));
>         String l; double t = 0; List<String> list = new ArrayList<>();
>         while ((l = r.readLine()) != null) {
>             String[] p = l.split(",");
>             if (p[2].equals("COMPLETED")) {
>                 double amt = Double.parseDouble(p[1]);
>                 if (amt > 100) { t += amt; list.add(p[0]); }
>             }
>         }
>         r.close();
>         PrintWriter w = new PrintWriter(new FileWriter(out));
>         w.println("Total: " + t);
>         for (String id : list) { w.println("Order ID: " + id); }
>         w.close();
>         System.out.println("Done processing " + f);
>     }
> }
> ```
>
> **Yêu cầu Vòng 1 (Robustness):** Bổ sung các kiểm tra sau:
> 1. Kiểm tra tham số `f` và `out` không null, không rỗng.
> 2. Dùng try-with-resources để tự động đóng `BufferedReader` và `PrintWriter`.
> 3. Bắt riêng `IOException` và `NumberFormatException`.
> 4. Nếu một dòng trong file CSV bị lỗi (sai định dạng, thiếu cột, amount không phải số), **bỏ qua dòng đó** và tiếp tục xử lý, không làm sập luồng.
> 5. Kiểm tra độ dài mảng `p.length >= 3` trước khi truy cập chỉ mục.
> 6. Xuất mã nguồn Java đã sửa.

### Vòng 2 (Clean Code & SRP)

> Dựa trên mã nguồn sau Vòng 1, hãy refactor tiếp:
>
> **Yêu cầu Vòng 2 (Clean Code & SRP):**
> 1. Tách lớp `ReportGenerator` thành 3 phương thức riêng biệt:
>    - `readOrders(String filePath) -> List<String[]>`: đọc file, parse CSV, trả về danh sách dòng.
>    - `processOrders(List<String[]> rawOrders) -> ProcessResult`: lọc COMPLETED, amt > 100, tính tổng.
>    - `generateReport(ProcessResult result, String outputPath)`: ghi file đầu ra.
> 2. Đổi tên biến sang CamelCase rõ nghĩa:
>    - `f` → `inputFilePath` / `filePath`
>    - `out` → `outputFilePath`
>    - `l` → `line`
>    - `p` → `columns` / `parts`
>    - `t` → `totalAmount`
>    - `list` → `completedOrderIds`
>    - `r` → `reader`
>    - `w` → `writer`
> 3. Tạo class `ProcessResult` (record) chứa `totalAmount` và `completedOrderIds`.
> 4. Thêm hằng số cho magic number `100` và trạng thái `"COMPLETED"`.
> 5. Xuất mã nguồn Java đã sửa.

### Vòng 3 (Logging & Context Tuning)

> Dựa trên mã nguồn sau Vòng 2, hãy refactor tiếp:
>
> **Yêu cầu Vòng 3 (Logging & Context Tuning):**
> 1. Tích hợp Lombok: thêm `@Slf4j` lên class, thay thế `System.out.println("Done processing " + f)` bằng `log.info("Done processing {}", filePath)`.
> 2. Thêm log ở các bước quan trọng: bắt đầu xử lý, số dòng đọc được, tổng tiền, kết thúc.
> 3. Nếu gặp dòng lỗi, ghi `log.warn("Skipping malformed line {}: {}", lineNumber, line)` thay vì im lặng bỏ qua.
> 4. Trích xuất dấu phẩy `","` thành hằng số `CSV_DELIMITER`.
> 5. Đảm bảo các tham số đường dẫn được truyền động qua tham số phương thức (không hardcode).
> 6. Xuất mã nguồn Java hoàn chỉnh cuối cùng.
> 7. Giả sử dự án đã có sẵn Lombok trong classpath.

## Phần 3: Mã nguồn Java hoàn chỉnh sau tối ưu (do AI sinh ra)

```java
import java.io.*;
import java.util.*;

import lombok.extern.slf4j.Slf4j;

@Slf4j
public class ReportGenerator {

    private static final String COMPLETED_STATUS = "COMPLETED";
    private static final double MIN_AMOUNT = 100.0;
    private static final String CSV_DELIMITER = ",";

    public void generateReport(String inputFilePath, String outputFilePath) throws IOException {
        if (inputFilePath == null || inputFilePath.isBlank()) {
            throw new IllegalArgumentException("inputFilePath must not be null or empty");
        }
        if (outputFilePath == null || outputFilePath.isBlank()) {
            throw new IllegalArgumentException("outputFilePath must not be null or empty");
        }

        log.info("Starting report generation: input={}, output={}", inputFilePath, outputFilePath);

        List<String[]> rawOrders = readOrders(inputFilePath);
        log.info("Read {} lines from input file", rawOrders.size());

        ProcessResult result = processOrders(rawOrders);
        log.info("Processed orders: totalAmount={}, completedCount={}", result.totalAmount(), result.completedOrderIds().size());

        generateReportFile(result, outputFilePath);
        log.info("Report generated successfully at {}", outputFilePath);
    }

    private List<String[]> readOrders(String filePath) throws IOException {
        List<String[]> orders = new ArrayList<>();
        int lineNumber = 0;

        try (BufferedReader reader = new BufferedReader(new FileReader(filePath))) {
            String line;
            while ((line = reader.readLine()) != null) {
                lineNumber++;
                if (line.isBlank()) {
                    continue;
                }

                String[] columns = line.split(CSV_DELIMITER);
                if (columns.length < 3) {
                    log.warn("Skipping malformed line {}: insufficient columns ({})", lineNumber, line);
                    continue;
                }

                orders.add(columns);
            }
        } catch (FileNotFoundException e) {
            log.error("Input file not found: {}", filePath, e);
            throw e;
        }

        return orders;
    }

    private ProcessResult processOrders(List<String[]> rawOrders) {
        double totalAmount = 0.0;
        List<String> completedOrderIds = new ArrayList<>();

        for (int i = 0; i < rawOrders.size(); i++) {
            String[] columns = rawOrders.get(i);
            try {
                String status = columns[2].trim();
                if (!COMPLETED_STATUS.equals(status)) {
                    continue;
                }

                double amount = Double.parseDouble(columns[1].trim());
                if (amount > MIN_AMOUNT) {
                    totalAmount += amount;
                    completedOrderIds.add(columns[0].trim());
                }
            } catch (NumberFormatException e) {
                log.warn("Skipping line {}: invalid amount format '{}'", i + 1, columns[1]);
            } catch (ArrayIndexOutOfBoundsException e) {
                log.warn("Skipping line {}: missing required columns", i + 1);
            }
        }

        return new ProcessResult(totalAmount, completedOrderIds);
    }

    private void generateReportFile(ProcessResult result, String outputPath) throws IOException {
        try (PrintWriter writer = new PrintWriter(new FileWriter(outputPath))) {
            writer.println("Total: " + result.totalAmount());
            for (String orderId : result.completedOrderIds()) {
                writer.println("Order ID: " + orderId);
            }
        }
    }

    public record ProcessResult(double totalAmount, List<String> completedOrderIds) {}
}
```
